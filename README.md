# 🏋️ MAXXGYM

A modern gym management web application built for trainers and clients to manage workouts digitally.

---

## 🚀 Features

- 👨‍🏫 Trainer & Client Authentication
- 📝 Assign workout plans digitally
- 📱 Mobile-friendly client dashboard
- 💪 Workout split management
- 🔥 Drag & Drop workout builder
- ☁️ Firebase Authentication & Firestore
- ⚡ Fast React + Vite frontend

---

## 🛠️ Tech Stack

### Frontend
- React + Vite
- Tailwind CSS
- React Router DOM

### Backend & Database
- Firebase Authentication
- Firestore Database

### Additional Libraries
- DnD Kit
- React Hot Toast
- Lucide React Icons
